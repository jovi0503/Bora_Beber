# **App Name**: Bora Delivery

## Core Features:

- Homepage: Display a header with the logo and main menu buttons (Home, Catalog, Offers, Cart, Login/Register), a prominent banner with a catchy phrase, and featured promotions.
- Product Catalog: Show a listing of products with image, description, price, and 'Add to Cart' button. Include categories (Beers, Wines, Spirits, Energy Drinks, Non-Alcoholic) and filters (Price, Brand, Beverage Type).
- Shopping Cart: Display added products with options to change quantity or remove items.  Show the order summary with the total value. Include a 'Checkout' button.
- Login/Registration: Provide a simple form for login/registration using email, password, and the option to sign in with Google.
- AI Product Suggestions: Suggest a personalized set of product recommendations based on customer purchase history or collaborative filtering with other user's purchases. A tool would consider factors such as purchase history, best selling items, and time of day when deciding which product to show. 
- Order Placement: Enable registered users to easily place delivery orders using a previously saved address, or optionally provide a new delivery address. If no address is saved, prompt for the address before continuing.

## Style Guidelines:

- Primary color: Bright yellow (#FFD700) for prominence and energy.
- Background color: Dark Gray (#1A1A1A) for contrast.
- Accent color: Orange (#FFB300) for key interactive elements like CTAs.
- Headline font: 'Poppins' (sans-serif) for a modern, high-contrast feel, used in headlines and banners. Body font: 'PT Sans' (sans-serif) for legibility, used in product descriptions and user info.
- Use custom designed icons of beverages (beer, wine, etc) that help give an unique visual identity to the website.
- Employ a fully responsive layout for seamless adaptation across desktop and mobile devices. Prioritize larger, rounded, and visually prominent buttons in yellow to enhance user interaction.
- Implement smooth transitions and subtle animations on button hovers and page navigation to enhance user experience.
